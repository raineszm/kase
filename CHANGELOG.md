## v0.2.0 - 2026-01-20
### Added
* punch command

Output the title line for a case in a format ready
to be passed to punch e.g.,

punch add -t 12:34 case : "$(kase punch)" : my notes

* import command

New cases can now be imported from CSV files using the case import command. These CSV files can be generated by exporting a Salesforce report.
